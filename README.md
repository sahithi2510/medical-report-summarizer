# 🩺 Medical Report Summarizer

An AI-powered healthcare document analysis system that extracts text from medical reports, generates concise summaries, detects abnormal laboratory values, explains medical terminology, and provides downloadable reports.

**Recently Upgraded:** The system now features **Enterprise-Grade Privacy** with automatic PII (Personally Identifiable Information) redaction, ensuring patient data is anonymized before AI processing.

Built using Python, Streamlit, OCR, NLP, Microsoft Presidio, and Transformer-based summarization.

---

## 🚀 Features

### 🔒 Privacy-First PII Redaction
* Automatically detects and redacts patient names, phone numbers, emails, and dates using **Microsoft Presidio** and **SpaCy**.
* Replaces sensitive data with tags like `<PERSON>` or `<DATE_TIME>` to ensure safe data handling.

### 📄 Multi-Format Medical Report Processing
Supports:
* PDF reports
* DOCX reports
* TXT files
* Scanned medical reports (PNG, JPG, JPEG)

---

### 🔍 OCR-Based Text Extraction
Uses EasyOCR to extract text from scanned reports and medical images.

Supported image formats:
* PNG
* JPG
* JPEG

---

### 🤖 AI-Powered Summarization
Uses Google's T5 Transformer model (`t5-small`) to generate concise summaries of lengthy medical reports.

Benefits:
* Faster report understanding
* Reduced reading time
* Improved accessibility

---

### 🧪 Medical Parameter Extraction
Automatically detects common laboratory parameters such as:
* Hemoglobin (Handles various spellings e.g., Hgb)
* WBC
* Platelets
* Blood Sugar / Glucose
* Creatinine
* Cholesterol
* HbA1c

---

### ⚠️ Abnormality Detection
Compares extracted values against reference ranges and classifies them as:
* Normal
* High (Highlighted in Red)
* Low (Highlighted in Red)

---

### 📚 Medical Glossary
Highlights medical terminology and provides patient-friendly explanations.

---

### 📊 Interactive Visualizations
Provides visual dashboards for:
* Laboratory values
* Health indicators
* Result distributions

Built using Plotly.

---

### 🔊 Audio Summaries & 📥 Export Options
* Converts generated summaries into speech for improved accessibility.
* Export summaries as PDF or TXT.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/medical-report-analyzer.git

cd medical-report-analyzer
```

### Create Virtual Environment

```bash
python -m venv venv
```

Windows:
```bash
venv\Scripts\activate
```

Linux/Mac:
```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Download SpaCy Language Model
Required for Microsoft Presidio PII redaction:
```bash
python -m spacy download en_core_web_lg
```

---

## ▶️ Running the Application

```bash
python -m streamlit run app.py
```

The application will open in your browser automatically.
